# globalSearch
i made this for myself but why not make it public for people to use, 
this plugin is to make discord's search a bit better and cooler, making it search everywhere for whatever you want saving you time from going to each server and searching, it also supports DMs.

go to C:\Users\Your user\Documents\Vencord\src\userplugins

then download the plguin or use git clone (git clone https://github.com/SAMURAI-99/globalSearch.git), that's all!
